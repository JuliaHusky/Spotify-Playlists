# Spotify Playlist Builder 

A simple **Python command-line application** that uses the **Spotify Web API** to search for tracks and build a custom playlist saved locally as a text file.

This project demonstrates API usage, authentication, user input handling, and file I/O in Python.

---

## Features

- Search for songs using the Spotify Web API  
- Retrieve song names and Spotify URIs  
- Build a playlist interactively via a menu  
- Save playlists locally as `.txt` files  

---

## Technologies Used

- Python 3  
- Spotipy (Spotify Web API wrapper)  
- Spotify Web API
